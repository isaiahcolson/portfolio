Featured Work.

Watch Later.

A web app for users to add movies and shows to their watchlist and then manually sort this list. I have noticed that there are no streaming services that offer manual ordering for user's watchlists, I wanted to solve this issue.

- Desktop and Mobile

- Built with: MongoDB, Express, React, Node.js

Nut Index.

A web app to track inventory. User’s can create multiple lists and items can be held in those lists. If an item is low on stock, the quantity will turn red alerting them to reorder. They will also be able to see it show up on their dashboard. Oh, and there’s dark mode.

- Desktop only

- Built with: MongoDB, Express, Node.js

Know This Trivia.

A web app trivia game. This is just a fun way to test your knowledge on movies and shows. There are ten categories and each category has 10 questions in it. See if you can get them all correct!

- Desktop only

- Built with: HTML, CSS, Javascript
